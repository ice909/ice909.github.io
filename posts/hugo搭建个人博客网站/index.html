    Hugo搭建个人博客网站

    September 8, 2023 · ice

    前言#

    Hugo 是一款基于 Go 语言而实现的静态网站生成器，具有简单易用、高效易扩展、快速部署的特点。 这里先给出 Hugo 的中英文官方文档，方便大家遇到问题时查阅：

    Hugo 中文文档 Hugo 英文文档

    接下来就来看看，如何利用 Hugo 来搭建一个博客。

    1. 安装Hugo#

    当你打开Hugo的Github Release页面后，你会看到各个版本的Hugo安装包。选择最新的版本是一个明智的选择，因为它通常包含了最新的功能和修复的bug。

    在选择版本后，你需要根据你当前的操作系统选择对应的安装包进行下载。Hugo提供了多种操作系统的安装包，包括Windows、macOS和Linux。确保选择与你的操作系统兼容的安装包。

    安装完成后，你可以在命令行中输入hugo version来验证Hugo是否成功安装并显示当前安装的版本号。如果成功显示版本号，则说明Hugo已经安装成功。

    $ hugo version
hugo v0.118.2-da7983ac4b94d97d776d7c2405040de97e95c03d linux/amd64 BuildDate=2023-08-31T11:23:51Z VendorInfo=gohugoio

    现在你已经成功安装了Hugo，你可以开始使用它来创建和管理静态网站了。

    2. 创建博客目录#

    初始化网站目录 $ hugo new site blog 初始化之后，得到的，目录结构如下:

    $ tree blog
blog
├── archetypes
│   └── default.md
├── assets
├── content
├── data
├── hugo.toml
├── i18n
├── layouts
├── static
└── themes

    这里需要注意的是hugo.toml,这个配置文件是用来配置Hugo的一些全局参数的。这个文件默认有以下三个配置:

    baseURL = 'https://example.org/'
languageCode = 'en-us'
title = 'My New Hugo Site'

    我不太喜欢toml的语法,所以我把它改成了hugo.yaml。

    baseURL: 'https://example.org/'
languageCode: 'en-us'
title: 'My New Hugo Site'

    3. 安装主题#

    可以去Hugo官网找到你想要的主题,这里我选择了PaperMod主题，然后利用如下命令将主题下载到项目根目录的themes文件夹中:

    git submodule add https://github.com/adityatelange/hugo-PaperMod themes/papermod

    4. 配置主题#

    修改hugo.yaml文件

    baseURL: 'https://example.org/'
languageCode: 'en-us'
title: 'My New Hugo Site'
theme: 'papermod' # 设置主题

    5. 预览网站#

    hugo server

    运行hugo server后，浏览器打开http://localhost:1313/就可以看到你的网站了。
    因为还没有添加文章，所以内容会是空白的

    总结#

    以上就是利用Hugo搭建个人博客网站的基本步骤。
    希望大家博客网站搭建好之后，也可以开始写作和分享自己的想法和经验。
